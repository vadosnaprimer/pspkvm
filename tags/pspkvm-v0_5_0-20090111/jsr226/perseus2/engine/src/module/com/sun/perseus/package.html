Perseus is an API for rendering SVG (Scalable Vector Graphics) content on the Java platform.

SVG is an XML syntax for describing rich, animated and scripted Graphics

.

Perseus can handle the SVG Tiny profile of SVG.

The Perseus packages are as follows:

  • com.sun.perseus.builder. Contains the classes used to convert an SVG Tiny document into an internal representation that can be rendered. This packages uses the SAX API and relies on the com.sun.perseus.parser package for parsing specific attribute types such as CSS color values.
  • com.sun.perseus.parser. Contains classes for parsing complex attribute values such as affine transformation, path data or color values.
  • com.sun.perseus.model. Contains the classes used to represent the SVG Tiny document in memory. This can be though of as a lightweight Document Object Model.
  • com.sun.perseus.render. Contains the classes used to render the internal memory representation of the SVG document.
  • com.sun.perseus.apps. Contains unsupported sample applications that use the Perseus API.
  • com.sun.perseus.util. Contains supporting utilities such as constants interfaces (e.g., SVGConstants which contains all the SVG Tiny constants.
